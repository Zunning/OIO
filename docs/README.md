# OIO Docs

这个目录用于存放 OIO 的开发计划、功能方案和开发日志。

## 文档索引

- `development-plan.md`：当前开发计划，记录第一版已完成功能和后续方向。
- `development-log.md`：开发日志，记录第一版从想法到完成的过程。
- `source-intent-highlight-plan.md`：原始输入高亮与 Source Intent 绑定方案。
- `review-experience-change-plan.md`：复习体验优化方案。
- `review-system-v2-plan.md`：复习系统 V2 方案，包括重要卡片、毕业、继续复习、总结和签到。
- `prompt-workflow-optimization-plan.md`：手动 AI 提示词工作流优化方案。

## 项目入口

项目根目录保留可运行文件：

```text
index.html
styles.css
app.js
```

直接打开 `index.html` 即可使用当前静态版 OIO。
