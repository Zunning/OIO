# OIO 使用说明

## 1. 打开方式

OIO 当前是本地静态版。

直接打开项目根目录里的：

```text
index.html
```

即可使用。

当前数据保存在浏览器的 `localStorage` 里，所以尽量固定使用同一个浏览器打开。

## 2. 基本学习流程

推荐日常流程：

```text
原始输入
-> 用 Prompt 发给 ChatGPT
-> 粘贴 OIO 格式结果并解析
-> 标出原始意图
-> 高亮英文 chunk
-> 匹配原始意图
-> 挖空生成卡片
-> 复习
```

## 3. 使用 Prompt 生成文本

进入 `Prompts` 页面。

可以选择：

- Everyday Expression / IELTS Training
- 英式英语 / 美式英语
- CEFR 难度或 IELTS 分数
- 是否允许 emoji
- dialogue / writing / work / casual / story 等风格

把自己的原始输入写进去后，点击：

```text
复制提示词 + 原始输入
```

再粘贴到 ChatGPT。

ChatGPT 返回 `:::OIO ... :::` 格式后，回到 OIO 新建文本窗口，粘贴到 `AI 输出`，点击：

```text
自动解析
```

## 4. 新建文本

点击：

```text
＋ 新建文本
```

可以填写或自动解析：

- 标题
- 文本类型
- 日期
- 原始输入
- 英文改写
- 中文翻译
- 备注
- 标签
- 图片

保存后，英文改写会自动复制到剪贴板。

如果有原始输入，保存后会进入原始意图高亮界面。

## 5. 原始意图

原始意图就是：

```text
我原来想表达，但不知道英文怎么自然说的地方
```

在原始输入里选中一小段内容，按 `Enter` 或点击保存，即可保存为原始意图。

如果选中的是英文单词的一部分，OIO 会自动扩展到完整单词。

## 6. 高亮英文 Chunk

进入文本详情页后，在英文改写里选中一个值得学习的表达块。

可以：

```text
选中表达 -> 按 Enter
```

或点击：

```text
挖空选中内容
```

保存 chunk 后，如果还有未匹配的原始意图，会先弹出匹配窗口。

选择对应的原始意图后，再进入挖空界面。

如果原始意图已经都匹配完了，会直接进入挖空界面。

## 7. 挖空生成卡片

在挖空界面里：

- 单击单词：选择 / 取消，用于多词挖空
- 双击单词：直接生成卡片并保存
- 保存按钮：用于多词表达

例如：

```text
in the mood for
on the fence about
get carried away
```

这类表达建议单击多个词后再保存。

生成卡片时可以勾选：

```text
重要
```

重要卡片会在复习排序中优先出现。

## 8. 复习

进入 `复习` 页面。

可以选择：

- 复习全部
- 只复习重要
- 每组数量：5 / 7 / 10 / 15 / 20

复习时：

```text
看挖空句
-> 输入答案
-> 检查
-> Again / Good / Easy
```

评分含义：

- Again：没掌握，10 分钟后再出现
- Good：掌握一次，明天再出现
- Easy：很熟，会拉长间隔

Good / Easy 累计 3 次后，卡片会毕业，不再进入普通复习。

## 9. 复习上下文

复习时可以看到：

- 当前挖空句
- 英文上下文
- 文本图片
- 原始意图
- 可选显示原始输入

上下文会隐藏答案，避免提前泄露。

## 10. 签到和复习记录

首页会显示：

- 今日复习
- 连续复习天数
- 签到日历

每天至少复习 1 张卡片，就会自动签到。

点击日历某一天，会先看到当天复习了哪些文本组。

点击文本组后，可以查看那一组复习过哪些 chunks。

## 11. 图片

每篇文本可以上传一张或多张图片。

图片适合保存：

- 截图
- 聊天记录
- 网页材料
- 书页
- 生活场景照片

图片会显示在：

- 文本详情页
- 新建 / 编辑文本窗口
- 复习上下文
- 导出内容

注意：当前图片保存在 `localStorage`，图片太多会让浏览器本地数据变大。

## 12. 搜索

文本库支持搜索。

搜索范围可以选择：

- 全部
- 标签
- 文本
- 标题

其中：

```text
全部 = 标题 + 原文 + 改写 + 翻译 + 备注 + 标签
文本 = 原文 + 改写 + 翻译 + 备注
```

## 13. 导出

文本库可以批量导出。

操作：

```text
勾选文本
-> 选择格式
-> 选择字段
-> 复制导出内容 / 下载文件
```

支持格式：

- Markdown
- HTML
- 纯文本

支持字段：

- 标题
- 日期
- 原文
- 改写
- 翻译
- 图片

下载文件规则：

- Markdown -> `.md`
- HTML -> `.html`
- 纯文本 -> `.txt`

如果需要导出图片，优先推荐 HTML。

## 14. 当前限制

当前版本仍然是本地静态版：

```text
index.html
styles.css
app.js
localStorage
```

限制：

- 没有云同步
- 没有自动备份
- 图片多了可能占用较多 localStorage
- 换浏览器可能看不到原来的数据

后续可以优先做：

- 数据备份 / 导入
- IndexedDB
- 按组导出
- 音频 / 听音练习
